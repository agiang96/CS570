Projects from CS570 - Operating Systems class:

Zero/  - a simple lexical analyzer that stores "words" to an array and special characters return a number.

One/   - a slightly more complex lexical analyzer that builds on Zero/ with more special character behaviors. 

Two/   - p2.c a simple shell that uses One/ to parse user input from the command line and can handle 1 pipe
       - p4.c a slightly more complex shell that builds on p2.c that can handle up to 10 pipes

Three/ - semaphore and mutex project that deal with "robots" that represent processes.
